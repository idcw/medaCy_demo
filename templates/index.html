Extracting Information from Medical Text with Machine Learning
MedaCy is a medical text mining framework we built to facilitate the engineering, training and application of machine learning models for medical information extraction. This demo provides an interface to the predictive functionality of a model trained to extract these entities from medical documents:

Drug, Form, Route, ADE, Reason, Frequency, Duration, Dosage, Strength, Dose, Active Ingredient, Volume of Distribution, Adverse Reaction, Trade Name, Duration
and more!
  • Clinical Notes is a machine learning trained to extract from text you might find in a physician's report for a patient visit.
  • Nanodrug Labels is machine learning model trained to extract from FDA Nanoparticle drug labels.
